# React Fish Store

## Title
React Fish Store

## Tech used
React.js, SASS/css, Javascript ES6, Node.js, Express.js

## How to install and run
- git clone https://github.com/peterfuwork/express-fish-store.git
- npm install
- npm run dev
- It runs on localhost:3001/

## Two json files
localhost:3001/fish
localhost:3001/comments
